# 🚀 LALALOU KPI Dashboard

Dashboard profesional para análisis de KPIs de stock con capacidad de upload de archivos Excel y visualización interactiva.

## ✨ Características

- 📤 **Upload de archivos Excel** con drag & drop
- 📊 **KPI Cards dinámicos** con métricas principales
- 📈 **Gráficos interactivos** (Chart.js)
- 📋 **Tablas de datos** con filtros y búsqueda
- 📅 **Comparación histórica** de períodos
- 📱 **Diseño responsive** para todos los dispositivos
- 🎨 **UI moderna** con Tailwind CSS

## 🎯 Demo

**Acceso directo:** Abrir `index.html` en tu navegador

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript, Alpine.js
- **Charts**: Chart.js para visualizaciones
- **Styling**: Tailwind CSS
- **Icons**: Heroicons
- **Data**: Estructura JSON optimizada para Excel

## 📊 Funcionalidades de KPI

### KPIs Principales
- Total Sales: Ventas totales del período
- Revenue Growth: Crecimiento vs período anterior
- Active Products: Productos activos en catálogo  
- Stock Value: Valor total de inventario

### Gráficos Incluidos
- **Line Chart**: Tendencia de ventas mensual
- **Bar Chart**: Top productos más vendidos
- **Pie Chart**: Distribución por categorías
- **Doughnut Chart**: Estado de stock

### Análisis de Datos
- Procesamiento automático de archivos Excel
- Cálculo de KPIs en tiempo real
- Comparación histórica de períodos
- Exportación de reportes

## 🚀 Uso

1. **Abrir** `index.html` en tu navegador
2. **Subir** archivos Excel usando drag & drop
3. **Visualizar** KPIs y gráficos actualizados
4. **Explorar** diferentes períodos y comparaciones
5. **Exportar** reportes según necesidad
